Top 3 Editing Tools for Photography


Adobe Lightroom interface
  • Sleek design
  • Very intuitive
  • Built in walkthroughs
  • Limited editing options
  • No layers
  • Very few graphic design options

Adobe Photoshop interface
  • Unlimited editing options
  • Uses layers and blending
  • Tools for graphic design
  • Not very intuitive
  • Clunky workflow
  • Not good for preparing many pictures at once

Luminar by Skylum interface
  • Same keyboard shortcuts as Lightroom
  • Can catagorize and sort photos
  • Has an AI slider that edits photos
  • Requires a lot of memory
  • Slow editing process
